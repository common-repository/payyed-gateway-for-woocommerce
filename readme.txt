== Description ==
Accept payments from MPESA, PayPal, Stripe, Intasend, Flutterwave, Credit Cards, Debit Cards via Payyed.org, we host 10+ payment gateways making it flexible to shift between any at will. This plugin fully relies on the services from payyed.org (where payment gateways are hosted). This is a woocomemrce payment plugin that you can rely on!  

With Payyed Gateway for WooCommerce, you can accept payments via:

* Credit/Debit Cards
* MPESA
* PayPal
* PayStack
* Stripe
* Authorize.net
* Intasend
* Payyed
* Flutterwave
* KopoKopo
* And more ++

= Getting Started =

* Go to https://payyed.org/ and create account using your name, email and phone contact
* Click settings and generate your private key
* Go back to your WordPress backend >> Woocommerce settings and enter the key generated
* Save and that is all!

== Installation ==

*   Go to __WordPress Admin__ > __Plugins__ > __Add New__ from the left-hand menu
*   In the search box type __Payyed Gateway for WooCommerce__
*   Click on Install now when you see __Payyed Gateway for WooCommerce__ to install the plugin
*   After installation, __activate__ the plugin.
*   Go to Payyed.org, login and get your API keys.


= Payyed Setup and Configuration =
*   Go to __WooCommerce > Settings__ and click on the __Payments__ tab
*   You'll see Payyed listed along with your other payment methods. Click __Set Up__
*   On the next screen, configure the plugin. There is a selection of options on the screen. Read what each one does below.

1. __Enable/Disable__ - Check this checkbox to Enable Payyed on your store's checkout
2. __Title__ - This will represent Payyed on your list of Payment options during checkout. It guides users to know which option to select to pay with Payyed. __Title__ is set to "Pay via Payyed" by default, but you can change it to suit your needs.
3. __Description__ - This controls the message that appears under the payment fields on the checkout page. Use this space to give more details to customers about what Payyed is and what payment methods they can use with it.

4. __API Keys__ - The next two text boxes are for your Payyed API keys, which you can get from your Payyed Dashboard.
 
8. Click on __Save Changes__ to update the settings.
